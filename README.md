# Hotel Management System

## Setup

### Requirements
- Java 17 or above
- Maven 3.8 or above

### Run
1. Clone the repo:
   ```bash
   git clone git@github.com:mithil27360/HMS-OSDL.git
   ```
2. Go to project folder:
   ```bash
   cd HotelManagement
   ```
3. Start the app:
   ```bash
   mvn clean javafx:run
   ```
